🎉 게임 완료 축하 모달 & 빵빠레 효과 사용자 경험(UX)

스도쿠 퍼즐을 성공적으로 풀어냈을 때, 사용자에게 확실한 피드백과 성취감을 줄 수 있도록 축하 팝업과 애니메이션을 도입했습니다.

  • 정답 판정 즉시 팝업 표시: 기존 메시지 텍스트 출력 방식에서 화면 전체 블러 처리와 함께 커스텀 모달 윈도우가 등장하도록 전환했습니다.
  • AI 생성 빵빠레 이미지: 나팔, 선물, 폭죽 등이 어우러진 화려하고 세련된 분위기의 축하 이미지가 모달 내에 나타납니다.
  • 소요 시간 출력: 스도쿠 해결에 걸린 시간을 분/초 단위로 계산하여 팝업창 중앙에 명확히 표기합니다.
  • 부드러운 애니메이션: 빵빠레 이미지가 위아래로 둥실둥실 움직이는 바운싱 효과가 구현되었습니다.

✓ 채워진 숫자 버튼 비활성화 (완료 배지) 편의성 개선

보드판에 9개가 모두 입력되어 더 이상 채워 넣을 필요가 없는 숫자를 시각적으로 흐리게 만들고 클릭을 차단합니다.

  • 실시간 숫자 세기: 힌트 숫자와 사용자 입력 숫자를 실시간으로 합산하여 특정 숫자가 9개 채워지는 것을 감시합니다.
  • 시각적 피드백: 완성된 숫자 버튼은 불투명도가 낮아지며(opacity: 0.25), 우측 상단에 초록색 완료 배지(✓)가 생성됩니다.
  • 동적 상태 복원: 중복 입력 등의 사유로 숫자가 9개 완성되었다가 지우기 혹은 수정으로 8개 이하가 되면 해당 숫자 버튼이 즉시 원상 복원됩니다.
  • 키보드 예외 허용: 마우스/터치 클릭은 차단하지만 물리 키보드 단축키(1~9) 입력은 정상 허용하여 숙련자들의 유연한 편집 동선을 보장합니다.
✏️ 연관 메모 자동 제거 및 되돌리기 복원 핵심 게임 로직

진짜 숫자를 확정 입력할 때, 연관된 셀의 임시 메모들을 자동 청소하여 불필요한 입력을 지워나가는 프로세스를 자동화했습니다.

  • 3방향 자동 메모 지우기: 확정 숫자가 위치한 가로줄(Row), 세로줄(Col), 3x3 박스 내의 동일한 숫자 메모들을 즉시 감지하여 안전하게 일괄 삭제합니다.
  • 완벽한 롤백(Undo) 지원: 단순 본인 칸의 값만 되돌리던 기존 스택을 다중 셀 정보 추적 방식으로 개편하여, `되돌리기` 실행 시 자동으로 지워졌던 주변 칸의 메모 세트까지 완벽 복구합니다.
  • 상태 저장 고도화: 새로고침 및 이탈 방지 상태 동기화 시, 확장된 다중 기록 스택 정보도 유실되지 않도록 JSON 직렬화(Serialization) 코드를 전면 개정하였습니다.
